# Remote Mouse

把你的手机变成电脑的局域网触控板和键盘。

Remote Mouse 会在本机启动一个 Web 服务，在终端显示二维码，手机扫码后即可在同一局域网内打开控制页面。页面采集鼠标移动、点击、滚动和文本输入，通过 WebSocket 发送到服务端，再由宿主机执行对应的鼠标和键盘操作。

## 功能特性

- 终端二维码快速入网访问
- 手机端触控板界面，支持左键、右键、双击、滚动
- 支持文本输入，以及 `Enter`、`Tab`、`Backspace` 快捷按键
- 6 位房间码 + 电脑端确认，避免局域网内被直接接管
- 仅本机可访问的 Host 管理页，可批准、拒绝、撤销设备授权
- 已补充自动化测试，覆盖配对和权限控制流程

## 环境要求

- Node.js 22+
- 手机和电脑位于同一局域网
- macOS 需要给终端应用或 Node.js 打开“辅助功能/Accessibility”权限

## 快速开始

```bash
npm install
npm start
```

启动后，终端会输出：

- 局域网访问地址
- 二维码
- 6 位房间码
- Host 管理页地址

## 使用步骤

### 1. 启动服务

```bash
npm start
```

如果 `3000` 端口被占用，可以改用其他端口：

```bash
PORT=3001 npm start
```

### 2. 在电脑上打开 Host 管理页

默认地址：

```text
http://127.0.0.1:3000/host
```

这个页面只允许在运行服务的电脑本机访问。

### 3. 在手机上打开控制页

- 直接扫描终端二维码，或
- 在手机浏览器里打开终端打印出来的局域网地址

### 4. 进行配对

- 在手机上输入终端或 Host 管理页显示的 6 位房间码
- 点击 `发起配对`
- 在电脑 Host 管理页点击 `允许`

完成后，手机就能控制电脑鼠标和键盘。

### 5. 撤销设备授权

在 Host 管理页的“已授权设备”区域，点击对应设备旁边的 `撤销授权`。

## 支持的控制方式

- 单指拖动：移动鼠标
- 双击触控区：触发双击
- 双指上下滑动：滚轮滚动
- 按钮区：左键 / 右键 / 双击
- 输入框：向当前电脑焦点窗口发送文本

## 测试

运行自动化测试：

```bash
npm test
```

当前测试覆盖：

- 相对鼠标移动逻辑

## 注意事项

- 远程控制本质上是在驱动宿主机输入接口，本机手动操作鼠标时仍然可能覆盖远程位置。
- 当前已改为“即时定位”而不是“动画式移动”，能减少与本机触控板抢控制权的问题。
- 如果启动时报 `EADDRINUSE`，说明端口已经被其他进程占用。
